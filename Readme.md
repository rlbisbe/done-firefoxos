Done for Firefox OS
=====================

Done es una aplicación de gestión de tareas, hecha durante un hackatón para probar Firefox OS.

Tecnologías empleadas
---------------------

- Firefox OS
- Knockout.js
- Gaia Building blocks
- CSS3 transforms
